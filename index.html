❓ FAQ

Frequently asked questions

Everything you might want to know about Slither Pro before you install.

Is Slither Pro free to use?

Yes. All core features — live in-game chat, 40+ skins, smart zoom, live HUD, minimap, death recap, animated name tags, custom food styling, grid overlays, and aura effects — are completely free, forever. No ads, no paywalls, no watermarks.

The optional Pro subscription ($2.99/month) unlocks the AI bot, auto-boost, and auto-play for players who want full automation, and starts with a free trial so you can try it risk-free.

Which browsers are supported?

Slither Pro runs on Chrome, Firefox, and Microsoft Edge. It uses the Manifest V3 standard, so any Chromium-based browser (Brave, Opera, Vivaldi, Arc) works too. Firefox requires version 142 or later.

Does Slither Pro collect any personal data?

No. We do not collect browsing history, keystrokes, clicks, IP addresses, or any personally identifiable information. Your preferences are stored locally on your device via the browser's built-in storage API.

The only data that leaves your device is the email address you provide when subscribing to Pro, which is handled securely by Stripe for payment verification.

Will using Slither Pro get my account banned?

Slither.io does not use persistent accounts, so there is no account to ban. Slither Pro runs entirely client-side in your browser — the game servers receive the same input they always do.

That said, always use game mods responsibly and respect other players.

How smart is the AI bot?

The bot uses a multi-goal scored decision engine. Each frame, it evaluates food value, nearby threats, kill opportunities, and escape routes — then picks the action with the highest combined score.

It includes 16-sector predictive collision avoidance, adaptive strategy that shifts as your snake grows, and a dedicated coil-survival system that detects trap patterns and calculates escape gaps.

How do I cancel my Pro subscription?

Cancel anytime through the Stripe billing portal, accessible directly from the extension popup. Your Pro access stays active until the end of your current billing period. No prorated refunds for partial months.

Does the AI bot work offline?

Short answer: no. Slither.io itself needs an internet connection, and so does Pro subscription verification.

Your Pro status is re-verified with our server every time you open the extension popup and every time you load slither.io. If the extension can’t confirm an active subscription for more than 24 hours, the bot automatically stops until you’re back online and verified. This keeps cancellations and refunds honest, and stops shared or leaked credentials from running indefinitely.

All other features — chat, skins, zoom, HUD, minimap, death recap — remain available regardless.

How are releases signed and verified?

Every release ZIP's SHA-256 fingerprint is published in a machine-readable manifest at downloads/checksums.json. Each download card links directly to it so you can verify the fingerprint of any copy you receive.

The extension also ships a per-file integrity manifest baked in at build time — the bot runtime hashes its own code on load and refuses to start if anything was tampered with.

Does Slither Pro work on mobile devices?

Slither Pro is a desktop browser extension. Mobile browsers do not fully support Manifest V3 extensions yet. Install Slither Pro on the desktop version of Chrome, Firefox, or Edge for the best experience.

How do I install Slither Pro?

Click the install button on the Chrome Web Store (or Firefox Add-ons if you're on Firefox), confirm the install, then open slither.io. The Slither Pro icon appears in your browser toolbar — click it to open the customization popup.
